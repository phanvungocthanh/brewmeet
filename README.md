# BrewMeet

BrewMeet is a coffee discovery and meetup app that helps users find cafés based on their needs, such as atmosphere, distance, price, coffee quality, and work-friendliness.

## Demo Video

[![Watch the BrewMeet demo](https://img.youtube.com/vi/jWr0Xd0sXik/0.jpg)](https://youtube.com/shorts/jWr0Xd0sXik)

## Current Features

- Landing page and onboarding
- User sign up and login screens
- Home screen with location and coffee recommendations
- Coffee shop search and discovery
- Meetups section
- Editable user profile
- Bottom-tab navigation
- React Native and Expo implementation

## Tech Stack

- React Native
- Expo
- Expo Router
- TypeScript
- JavaScript
